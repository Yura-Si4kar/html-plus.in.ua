Задача. Компания по транспортировке мебели заносит любую мебель на 1-й этаж бесплатно, со 2-го по 4-й включительно - по 20грн., с 5-го по 7-й включительно - по 30грн., а с 8-го и выше - по 40грн. Узнайте у пользователя, на какой этаж нужно занести мебель, и рассчитайте стоимость транспортировки. Это значит, что занос на 4-й этаж будет стоить 60грн, на 7-й - 150 грн, а на 10-й - 270 грн.
